CrystalCollector

Crystal Collector is a game in which on each refresh a new crystal is generated and each of the crystal is given a random number . There is a target to be achieved and it should be achieved by click on the image. If you exceed the target you lose and a new set of crystals are generated automatically wiht different points on each of them.Depending upon the play the win and the lose count updates automatically

Train Time

The project is called as Train Time. It asks the user to enter the details of the train and populated the DOM accordingly. This project makes use of Moment.js which is a javascript library for calculating time.
The project consists of usage of firebase for storing various parameters of the train including train name,start time,destination and the time off arrival according to the frequency of the train mentioned.
